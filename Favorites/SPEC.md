## 1. 프로젝트 목표

최신 제약·의약품 뉴스를 **한 눈에 쉽게 확인**할 수 있는 단일 HTML 페이지 대시보드 구축.

- 백엔드 없이 순수 HTML/CSS/JS로 동작
- 브라우저에서 바로 열어 사용 (파일 기반)
- 국내 기관(식약처·복지부·심평원)과 해외 기관(FDA·EMA·WHO) 기사를 통합 제공
- 사용자 상호작용(즐겨찾기, 메모, 읽음 처리 등)은 localStorage에 저장

## 4. 기능 명세

### 4-1. 탭 필터
- **전체 / 시장 전망 / 의약품 출시 / 정책·규제** (국내 통합)
- **FDA / EMA / WHO** (해외)
- 각 탭에 기사 수 표시, 미읽음 기사가 있으면 빨간 점(●) 표시

### 4-2. 날짜 범위 필터
- 전체 / 최근 6개월 / 3개월 / 1개월

### 4-3. 그룹 토글
- **연도별**: 2026년 / 2025년 섹션 구분
- **출처별**: 언론사·기관별 그룹
- **이슈별**: 9개 이슈 클러스터로 분류 (아래 4-7 참조)

### 4-4. 검색
- 기사 제목(국문+영문) 및 출처 대상 실시간 검색
- 검색어 하이라이트 (`<mark>` 태그)
- 검색어 초기화(✕) 버튼

### 4-5. 카드 기능
| 기능 | 동작 |
|---|---|
| 읽음 처리 | 기사 클릭 시 자동 처리, 카드 반투명 처리 |
| 즐겨찾기 ★ | localStorage 저장, 즐겨찾기 섹션에 상시 노출 |
| 메모 ✏️ | 카드별 메모 입력/저장/삭제, localStorage 저장 |
| 클립보드 복사 📋 | 기사 제목 + URL을 클립보드에 복사, 토스트 알림 |
| NEW 뱃지 | `isNew:true` 기사에 빨간 NEW 뱃지 + pulse 애니메이션 |
| Featured 카드 | 각 그룹의 첫 번째 기사가 "★ 주목 기사" 강조 표시 (전체 너비) |
| 교차 이슈 태그 | 이슈별 보기에서 다른 클러스터에도 해당되는 기사에 `↗ 태그` 표시 |

### 4-6. 키워드 트렌드 시각화
- 기사 제목 기준 10개 키워드 등장 빈도 수평 바 차트
- 기사 수 순 정렬, 0건 키워드 자동 숨김

### 4-7. 이슈별 그룹 (9개 클러스터)
| 이슈 | 주요 키워드 |
|---|---|
| GLP-1 · 비만치료제 | GLP-1, 비만치료, Wegovy |
| ADC 항암제 | ADC, 항체약물, Enhertu |
| CDMO · 위탁개발 | CDMO, 삼성바이오, 바이오로직스 |
| AI · 디지털 신약 | AI, 인공지능, 디지털치료 |
| 건강보험 · 약가 | 건강보험, 약가, 급여, 건정심 |
| 관세 · 글로벌 규제 | 관세, 트럼프, tariff |
| K-바이오 · 시장 | K-바이오, 바이오시밀러 |
| FDA 승인 동향 | FDA, 승인, approval |
| EMA · WHO 정책 | EMA, WHO, 팬데믹 |

- **중복 방지**: 각 기사는 첫 번째 매칭 그룹에만 배정
- **교차 태그**: 2개 이상 그룹에 해당하는 기사에 보조 이슈 태그 표시

### 4-8. 정책 변화 섹션 (상시 표시)
- 식약처·복지부·심평원 최신 5건 리스트 형태 표시
- 접기/펼치기 버튼, 상태 세션 유지

### 4-9. 해외 기관 섹션 (상시 표시)
- FDA·EMA·WHO 최신 6건 리스트 형태 표시
- 국문 제목 + 원문 영어 제목 함께 표시
- 접기/펼치기 버튼, 상태 세션 유지

### 4-10. 필터 상태바
- 툴바 아래에 상시 표시
- 현재 활성 필터(키워드 칩, 기간 칩, 검색어 칩) 표시
- 필터링된 기사 수 및 미읽음 기사 수 표시

### 4-11. 스크롤 위치 복원
- 페이지 스크롤 위치를 sessionStorage에 지속 저장
- 새로고침 시 이전 스크롤 위치로 자동 복원

### 4-12. 섹션 접기/펼치기
- 정책 변화 섹션, 해외 기관 섹션에 적용
- 접힘 상태는 sessionStorage에 저장하여 새로고침 후에도 유지
- 접힘 시 화살표 아이콘 회전 애니메이션

## 5. 데이터 구조

```js
{
  id:      'a01',           // 고유 ID
  date:    '2026-03-13',   // yyyy-mm-dd
  kw:      'market',       // 키워드 ID
  isNew:   true,           // NEW 뱃지 여부 (선택)
  titleKo: '국문 제목',    // 항상 필수
  titleEn: 'English title',// 해외 기사만 (선택)
  url:     'https://...',  // 기사 원문 링크
  source:  '언론사명',     // 출처
}
```

### localStorage 키
| 키 | 내용 |
|---|---|
| `ph_bm` | 즐겨찾기 기사 ID 배열 |
| `ph_rd` | 읽음 처리된 기사 ID 배열 |
| `ph_memo` | 기사별 메모 객체 `{ id: "메모 내용" }` |

### sessionStorage 키
| 키 | 내용 |
|---|---|
| `scroll_y` | 페이지 스크롤 Y 위치 |
| `sec_policy-sec` | 정책 섹션 접힘 여부 (`'1'`/`'0'`) |
| `sec_intl-sec` | 해외 섹션 접힘 여부 (`'1'`/`'0'`) |

## 7. 자동 업데이트 시스템

### 7-1. 업데이트 주기
- **매일 오전 8시 (KST)** GitHub Actions가 자동 실행
- `workflow_dispatch`로 GitHub Actions 탭에서 수동 즉시 실행 가능

### 7-2. 파일 구조
```
pharma-news/
├── news.html                          # 메인 페이지
├── scripts/
│   ├── update_news.py                 # RSS 수집·업데이트 스크립트
│   └── requirements.txt               # Python 의존성
└── .github/
    └── workflows/
        └── daily_update.yml           # GitHub Actions 워크플로우
```

### 7-3. 자동화 흐름
```
매일 08:00 KST
      │
      ▼ GitHub Actions (ubuntu-latest, Python 3.11)
scripts/update_news.py 실행
      │
      ├─ 1. RSS 수집 (국내 6개 + 해외 3개 피드)
      ├─ 2. 관련성 키워드 필터링
      ├─ 3. kw 자동 분류 (제목 기반)
      ├─ 4. 해외 기사 영문 → 국문 번역 (Google Translate)
      ├─ 5. 중복 제거 (URL + ID 해시 기반)
      ├─ 6. isNew 만료 처리 (14일 이상 → isNew 제거)
      └─ 7. news.html ARTICLES 배열 상단 삽입 → 커밋·푸시
```

### 7-4. RSS 수집 소스
| 분류 | 소스 | kw |
|---|---|---|
| 국내 | 식품의약품안전처 | mfds |
| 국내 | 메디게이트뉴스 | auto |
| 국내 | 팜뉴스 | auto |
| 국내 | 히트뉴스 | auto |
| 국내 | 약사공론 | auto |
| 국내 | 바이오타임즈 | auto |
| 해외 | FDA (신약 승인 + 보도자료) | fda |
| 해외 | EMA | ema |
| 해외 | WHO | who |

- `kw: 'auto'` 소스는 기사 제목을 분석해 mfds/mohw/hira/launch/market 중 자동 분류

### 7-5. 자동 생성 기사 ID 규칙
- 형식: `r` + URL MD5 해시 앞 7자리 (예: `r3f8a2c1`)
- 수동 입력 기사: `a01`, `b01` 등 알파벳+숫자 조합
- 중복 방지: 동일 URL은 항상 동일 ID → 재수집해도 중복 삽입 없음

### 7-6. 수동 보완 방법
1. **중요 기사 수동 추가**: Claude Code에 "이 기사 추가해줘 + URL" 요청
2. **잘못된 기사 제거**: news.html ARTICLES 배열에서 해당 항목 삭제
3. **번역 품질 수정**: `titleKo` 필드 직접 수정
4. **즉시 실행**: GitHub → Actions 탭 → `제약시장 뉴스 자동 업데이트` → Run workflow

| `deep-translator` | 영문 → 국문 번역 |
| `requests` | HTTP 요청 |
| `python-dateutil` | 날짜 파싱 |
